## 🎨 Design Philosophy

#### ⚙️ The Digital-Physical Bridge
> Finding the exact moment code escapes the screen. I optimize for latency, hardware constraints, and seamless interaction. If it breathes, moves, or interacts with the physical world, I’m building it.

#### 🏎️ High-Performance Mechanics
> Mastering optimization under strict environmental constraints, analyzing split-second decision making, and learning how to safely push systems to their absolute limits.

#### 📱 Tangible Human-Computer Interfaces  
> User interfaces should not just be flat pixels acting like posters behind glass. My obsession is designing experiences that feel substantial, heavy, and mechanically satisfying to touch.
